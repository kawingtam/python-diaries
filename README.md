# 🐍 Python Diaries

A personal Python practice repository documenting my learning journey from beginner-level syntax to problem-solving with data structures and algorithms.

This project started when I first began learning Python and became a space for me to practice consistently, build confidence, and track my progress over time.

---

## ✨ About This Repository

`python-diaries` contains a collection of small Python exercises, coding practice problems, and beginner-friendly projects. The goal of this repository is to show my growth in programming fundamentals, logical thinking, and algorithmic problem-solving.

Rather than being one large application, this repo is organized like a coding journal — each file represents a concept, exercise, or problem I worked through while strengthening my Python skills.

---

## 📚 Topics Practiced

- Python basics and syntax
- User input and conditionals
- Loops and functions
- Lists, sets, and dictionaries
- Number and star patterns
- Basic object-oriented programming
- Simple game logic
- LeetCode-style problem solving
- Depth-First Search
